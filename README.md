# 说明
本科毕设内容，基于元胞自动机和蒙特卡洛算法，模拟了电子在激光场中的发射和运动，从而实现激光场对电子控制。

main.m 是主程序，利用元胞自动机模拟电子发射与运动（其中包括 1 个主函数，7 个子函数）；

ADK.m 程序是利用ADK计算电离率，用于计算电离几率。

注意：在利用元胞自动机模拟中，蒙特卡洛算法模拟电离时，用了S型函数进行简化电离率。
